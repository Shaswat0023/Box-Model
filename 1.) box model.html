Box model

  • The Box model is the basic building block of CSS
  • According to the box model concept every element on a page is a rectangular box and may have width , height, padding, borders and margins

    • Suppose we take an image of a star and lets apply border to it and see what happens.

    We see that the border applied to star is in a form of a box.

    Now if we apply padding to our image then lets see what happens

    We see that the image is inside a rectangularbox and this is what box model says that every element on the page is inside a rectangular box
